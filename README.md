# 🚀 Prince Verified Portfolio & Certificate Validation System

## 📌 Overview

This is a full-stack web application that combines:

* Personal Portfolio
* Certificate Verification System
* Admin Panel

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL

## ✨ Features

* Portfolio showcase
* Certificate verification using unique ID
* Admin panel for uploading certificates
* Secure login system
* Responsive modern UI

## 🔍 Certificate Verification

Users can verify certificates using:

```
verify.php?cert_id=PRINCE-001
```

## 🧑‍💻 Admin Panel

* Upload certificates
* Manage records
* Delete certificates

## 📂 Project Structure

* index.php → Homepage
* verify.php → Verification system
* admin/ → Admin panel
* db.php → Database connection

## 🔐 Security

* Prepared statements
* Password hashing
* Input validation

## 🚀 Deployment

1. Upload files to hosting (InfinityFree)
2. Create MySQL database
3. Import SQL file
4. Update db.php credentials
